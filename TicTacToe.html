Tic Tac Toe Versions

Version 1, 6/17/2017

Basic Tic Tac Toe game playable by two people.

Link to TicTacToe Ver. 1

Version 2, 6/18/2017

Tic Tac Toe game with ability to be played by one or two people. One player mode is a game between the player and computer were the computer's choice is made at random.

Link to TicTacToe Ver. 2

Version 3, 6/20/2017

Tic Tac Toe game playable by one or two people, but changed prompt feature for user inputs to click and submit. During the process of changing from prompt to user input by clicks, there is additional logic necessary to fix error/bugs in the current programs logic. Issues include:

  • Players can override previous players moves.
  • After game is finished computer still makes a move
  • After game is finished user can still make a move
  • Additional errors/bugs to be fixed with further analysis
Link to TicTacToe Ver. 3

Version 4, 6/22/2017

Fixed errors/bugs from Version 3.

Link to TicTacToe Ver. 4

Version 5, 6/25/2017

Added two new difficulty modes where,

  • Easy difficulty has the AI chooses a random location.
  • Medium difficulty has the AI look to see if the user can win and blocks the user. Otherwise the computer chooses a random location.
  • Hard difficulty has the AI look to see if the computer can win. If the computer cannot win it will block the player from winning. Otherwise the computer chooses a random location.
Link to TicTacToe Ver. 5
