Tic Tac Toe Game

The game of Tic Tac Toe was chosen as a first project to learn how to link html, css, and js files together. This eventually processed into learning more html, css, and js techniques to improve the user interface of the web application.

List of goals accomplished and future goals:

  1. - Basic Framework
  2. - Add Basic AI for single player to play against
  3. - Remove messages and make game click based
