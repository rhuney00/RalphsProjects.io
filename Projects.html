Tic Tac Toe Game

The game of Tic Tac Toe was chosen as a first project to learn how to link html, css, and js files together. This eventually processed into learning more html, css, and js techniques to improve the user interface of the web application.

List of goals accomplished and future goals:

  • Basic Framework (Finished in Ver. 1)
  • Add Basic AI for single player to play against (Finished in Ver. 2)
  • Remove messages and make game click based (Finished in Ver. 4)
  • Add ability to have different difficulty AI with up to three different difficulties
  • Fix up appearance to make game more visually appealing

Version 1, 6/17/2017

Basic Tic Tac Toe game playable by two people.

Tic Tac Toe game with ability to be played by one or two people. One player mode is a game between the player and computer were the computer's choice is made at random.

Tic Tac Toe game playable by one or two people, but changed prompt feature for user inputs to click and submit. During the process of changing from prompt to user input by clicks, there is additional logic necessary to fix error/bugs in the current programs logic. Issues include:

  • Players can override previous players moves.
  • After game is finished computer still makes a move
  • After game is finished user can still make a move
